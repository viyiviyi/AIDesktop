# 应用图标和背景图管理

## 功能设计

### 存储
- 所有文件保存到 `apps_data/{appId}/media/` 目录下
- 每个应用最多保留 10 个图标和 10 个背景图
- 文件名格式: `icon_{timestamp}.png` / `bg_{timestamp}.png`
- 通过 `/api/files/{appId}/media/icon_xxx.png` 访问

### API
1. **上传图标/背景图**
   `POST /api/apps/:appId/media`
   body: FormData { type: 'icon'|'background', file: File }
   返回: { url: "/api/files/{appId}/media/icon_xxx.png", allIcons: [...], allBackgrounds: [...] }

2. **获取媒体列表**
   `GET /api/apps/:appId/media`
   返回: { icons: string[], backgrounds: string[] }

3. **删除媒体**
   `DELETE /api/apps/:appId/media`
   body: { url: "/api/files/..." }

4. **设置当前图标/背景**
   通过 PUT /api/apps/:appId 更新 meta 中的 icon/backgroundImage 字段
   值为 `/api/files/{appId}/media/icon_xxx.png` 或 `__reset__` 还原默认
   前端在上传成功后自动调用 updateApp 设置

### 前端
1. 图标点击 → 打开文件选择器（图片）
2. 选择后前端裁切（用 canvas 做正方形裁切）
3. 上传到服务端
4. 上传成功后显示在图标/背景选择列表中
5. 点击列表中的图可切换
6. 鼠标悬停显示删除按钮
