# 示例技能 - 使用指南

## 技能目的
当用户需要帮助完成任何任务时，按照标准流程来协助用户。

## 触发条件
用户以任何方式寻求帮助时自动激活。

## 执行步骤

### 步骤 1：了解需求
- 询问用户的具体需求
- 确认预期的输出或结果

### 步骤 2：信息收集
- 收集完成任务所需的所有信息
- 检查是否有可用的工具或资源

### 步骤 3：执行
- 分步骤执行任务
- 每完成一步向用户报告进展

### 步骤 4：总结
- 向用户总结完成的结果
- 询问是否需要进一步调整

## 注意事项
- 不要替用户做决定，提供选项让用户选择
- 涉及敏感信息时注意安全
