# AIDesktop 开发指引

> **📖 先看开发指南**: `docs/开发维护指南.md` — 包含架构说明、数据模型、关键业务流程和常见开发任务。

## 快速参考

- **端口**: 后端 27135，前端 5173（Vite 代理）
- **启动**: `cd server && npm run dev` + `cd client && npm run dev`
- **构建发布**: `./scripts/build-dist.sh`
- **数据路径**: `server/desktop_data/`
- **Agent 引擎**: pi-agent-core (vendor/pi/)
- **WebSocket**: `/api/ws`，事件通过 EventBus 广播
- **禁止自动 git commit**: 所有改动需经开发者确认
